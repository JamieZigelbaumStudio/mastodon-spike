# mastodon-spike
This is a repository containing the code for a webapp to interact with Mastodon

#Installation

[Yarn](https://yarnpkg.com/lang/en/docs/install/), or, [Npm](https://nodejs.org/en/)

# Start the app

To start up the application: 
```bash
$ yarn start
```

To stop the application:
```bash
yarn stop
```
